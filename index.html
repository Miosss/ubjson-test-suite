About

Currently (alpha-version) UBJSON test suite contains two online tools:

  • JSON to UBJSON converter (json-converter.html)
    You can use hash URLs to load JSON-encoded data.
  • Block-notation parser (blocks-parser.html)
    Load data example.

Specification

Latest specification can be found in official site: http://ubjson.org/.
